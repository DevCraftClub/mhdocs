# PersonSex

**Описание:** Enum для пола персон
Этот enum содержит все возможные значения пола персон, которые могут быть
возвращены API Kinopoisk.dev

## `getRussianName()`

**Описание:** Возвращает название пола на русском языке

## Cases

### `MALE`

**Значение:** `'male'`

### `FEMALE`

**Значение:** `'female'`
