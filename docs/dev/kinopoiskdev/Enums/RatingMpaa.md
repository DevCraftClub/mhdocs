# RatingMpaa

**Описание:** Enum для рейтингов MPAA
Этот enum содержит все возможные рейтинги MPAA, которые могут быть
возвращены API Kinopoisk.dev

## `getDescription()`

**Описание:** Возвращает описание рейтинга MPAA

## Cases

### `G`

**Значение:** `'g'`

### `PG`

**Значение:** `'pg'`

### `PG13`

**Значение:** `'pg13'`

### `R`

**Значение:** `'r'`

### `NC17`

**Значение:** `'nc17'`
